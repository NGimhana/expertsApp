Phase 2: Ranking Code Snippets by Comprehensibility

In this phase, you will review all 8 code snippets and rank them based on how easy or difficult they are to understand.

  1. Rank the snippets: Drag and drop each snippet into one of the 8 ranking positions below.
  2. Interpret the scale:
    • Position 1 (leftmost): Most comprehensible snippet.
    • Position 8 (rightmost): Least comprehensible snippet.
  3. Equal ranking: You may place multiple snippets in the same slot if they are equally comprehensible.
  4. Preview feature: Use the checkbox on each card to view the full code below.

Tip: You can revise your rankings at any time before proceeding to the next phase.
